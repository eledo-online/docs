# Statistical Functions
# SUM(V, X)
Sum of expression X on vector V

```
SUM( items, line_total )
```

## SUMC(V, X, F, T)
Conditional sum of expression X on vector V, where expression F has target value T

```
SUMC( items, line_total, type, "service" )
```
