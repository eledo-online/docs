# Components

Components allow you to insert dynamic content and logic into your document templates.

They are evaluated during document generation and replaced with real data.

---

## What are Components

Components are the building blocks of a template.

They let you:

- display data ([text](./text-box.mdx), [numbers](./text-box.mdx), [images](./dynamic-image.mdx))
- control visibility ([conditions](./block-condition.mdx))
- repeat content ([lists](../editor-tools/lists.mdx), [tables](../document-elements/table.mdx))
- apply logic directly in the document

---

## Where to find Components

Most components are available directly in the editor toolbar.

<ImageWithCaption
  src="/img/product/template-editor/components/toolbar.png"
  alt="Components Toolbar"
  caption="Components available in the editor toolbar."
/>

To insert a component:

1. Place the cursor where you want the component  
2. Click the corresponding icon in the toolbar  
3. Configure it and confirm  

The component will be inserted at the cursor position.

---

## Available components

Common components include:

- [**Text Box**](./text-box.mdx) — display dynamic values  
- [**Block Repetition**](./block-repetition.mdx) — repeat content for arrays  
- [**Block Condition**](./block-condition.mdx) — control visibility  
- [**Dynamic Image**](./dynamic-image.mdx) — insert images from data  
- [**Dynamic Checkbox**](./dynamic-checkbox.mdx) — display boolean values  
- [**Dynamic Link**](./dynamic-link.mdx) — generate clickable links  
- [**Dynamic Color**](./dynamic-color.mdx) — apply styles dynamically
- [**Barcodes**](./barcodes/index.mdx) — insert machine-readable barcode elements into your document
- [**Variables**](./variables.mdx) — define reusable values inside your templates  

---

## Components vs Form Components

There are two types of components in Eledo:

- **Components** — evaluated during document generation  
- [**Form Components**](../form-components/index.mdx) — filled in by the end user after generation  

Use Components when your data comes from your system.  
Use Form Components when input is expected from the document recipient.
