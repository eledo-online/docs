# Fonts & Styles

Fonts & Styles allow you to control the visual appearance of text in your document.

---

## Where to find Fonts & Styles

Font and style options are available in the editor toolbar.

<ImageWithCaption
  src="/img/product/template-editor/editor-tools/fonts-styles-toolbar.png"
  alt="Fonts and styles toolbar"
  caption="Font and style controls in the toolbar."
/>

---

## Available options

### Styles

Provides predefined text styles.

- Includes block styles and inline styles  
- Allows quick application of consistent formatting  

---

### Format (Paragraph format)

Controls the structure of the text.

- Normal text  
- Headings (e.g. Heading 1, Heading 2)  

This defines the role of the text within the document.

---

### Font

Changes the font family of the selected text.

---

### Size

Controls the font size.

---

### Line Height

Adjusts spacing between lines of text.

---

## How it works

- Select the text you want to modify  
- Choose the desired option from the toolbar  
- Changes are applied immediately  

---

## When to use Fonts & Styles

Use Fonts & Styles when you need to:

- control typography and readability  
- define document hierarchy (headings, sections)  
- apply consistent visual styling  

---

## Important notes

- **Format (paragraph format)** affects document structure  
- **Styles** apply predefined visual formatting  
- **Font and Size** adjust appearance directly  

These options can be combined for precise control.
