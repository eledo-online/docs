# Alignment

Alignment controls how content is positioned horizontally within the document.

---

## Where to find Alignment

Alignment options are available in the editor toolbar.

<ImageWithCaption
  src="/img/product/template-editor/editor-tools/alignment-toolbar.png"
  alt="Alignment toolbar icons"
  caption="Alignment options in the toolbar."
/>

---

## Available options

You can align content in four ways:

- **Left** — aligns content to the left side (default)  
- **Center** — centers content horizontally  
- **Right** — aligns content to the right side  
- **Justify** — spreads text evenly across the full width  

---

## How it works

- Select the text or element you want to align  
- Click the desired alignment option in the toolbar  
- The alignment is applied immediately  

---

## When to use Alignment

Use Alignment when you need to:

- adjust text positioning  
- improve layout and readability  
- create visually balanced sections
