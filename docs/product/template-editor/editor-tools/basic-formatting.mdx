# Basic Formatting

Basic Formatting allows you to apply simple text styling to your content.

---

## Where to find Basic Formatting

Formatting options are available in the editor toolbar.

<ImageWithCaption
  src="/img/product/template-editor/editor-tools/basic-formatting-toolbar.png"
  alt="Basic formatting toolbar"
  caption="Basic formatting options in the toolbar."
/>

---

## Available actions

### Bold

Makes the selected text bold.

---

### Italic

Applies italic styling to the selected text.

### Text color

Changes the color of the selected text.

---

### Background color

Applies a background highlight color to the selected text.

---

### Remove format

Removes all formatting from the selected text and resets it to default styling.

---

## How it works

- Select the text you want to format  
- Click the desired formatting option in the toolbar  
- The formatting is applied immediately  

---

## When to use Basic Formatting

Use Basic Formatting when you need to:

- emphasize important text  
- improve readability  
- clean up inconsistent formatting
