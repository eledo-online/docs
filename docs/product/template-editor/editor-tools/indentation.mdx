# Indentation

Indentation controls the horizontal spacing of content from the left margin.

It can be used to visually nest or offset content inside the document.

---

## Where to find Indentation

Indentation options are available in the editor toolbar.

<ImageWithCaption
  src="/img/product/template-editor/editor-tools/indentation-toolbar.png"
  alt="Indentation toolbar icons"
  caption="Increase and decrease indentation in the toolbar."
/>

---

## Available actions

### Increase indent

Moves the content at the cursor position further to the right.

---

### Decrease indent

Moves the content back toward the left margin.

This option is available only when the current indentation is greater than zero.

---

## How it works

- Place the cursor inside the content you want to adjust  
- Or select a block of content  
- Click the indentation option in the toolbar  
- The indentation is applied immediately  

---

## When to use Indentation

Use Indentation when you need to:

- visually separate nested content  
- improve readability  
- create structured layouts within text sections  

---

## Important notes

- Indentation works based on the cursor position — selecting content is optional  
- Decrease indent is available only when indentation has already been applied
