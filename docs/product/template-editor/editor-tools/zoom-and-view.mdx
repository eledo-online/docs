# Zoom & View

Zoom & View tools control how the document is displayed in the editor.

They affect only the visual representation and do not change the document content.

---

## Where to find Zoom & View

Zoom options are available in the editor toolbar.

<ImageWithCaption
  src="/img/product/template-editor/editor-tools/zoom-toolbar.png"
  alt="Zoom toolbar"
  caption="Zoom controls in the toolbar."
/>

---

## Available options

### Zoom

Adjusts the zoom level of the document in the editor.

- allows you to zoom in for detail  
- or zoom out for a broader view  

This helps when working with large or complex layouts.

---

### Maximize

Expands the editor to use more screen space.

- improves focus  
- provides more room for editing  

---

## How it works

- Select a zoom level from the dropdown  
- The document view updates immediately  
- Use maximize to expand the working area  

These changes affect only the editor view, not the generated document.

---

## When to use Zoom & View

Use Zoom & View when you need to:

- inspect details of your layout  
- work with complex templates  
- improve editing comfort  

---

## Important notes

- Zoom does not affect the final PDF output  
- It is only a visual aid inside the editor
