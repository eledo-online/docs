# Editor Tools

Editor Tools provide additional functionality for working with content inside the Template Editor.

They help you format text, control layout, and fine-tune how your document is structured.

---

## What you will find here

This section covers the tools available in the editor toolbar and how they affect your document.

Topics include:

- **Basic Formatting** — text styling (bold, italic, etc.)  
- **Lists** — ordered and unordered lists  
- **Alignment** — horizontal alignment of content  
- **Indentation** — spacing and structure  
- **Fonts & Styles** — typography and visual appearance  
- **Source Mode** — editing the underlying HTML  
- **Paste Options** — controlling how content is inserted  
- **Zoom & View** — adjusting how the document is displayed  

---

## How it fits into the Template Editor

Editor Tools operate directly on the content in your document.

- They do not define data or logic  
- They control how content looks and behaves  
- They work alongside components and document elements  

---

## Show Blocks

Show Blocks is a toggle that displays the structure of the document.

<ImageWithCaption
  src="/img/product/template-editor/editor-tools/show-blocks.png"
  alt="Show blocks enabled"
  caption="Paragraph blocks visualized in the editor."
/>

When enabled:

- paragraphs are outlined as blocks  
- each block shows a small **P** marker  
- the document structure becomes easier to see  

---

### When to use Show Blocks

Use this feature when you need to:

- understand document structure  
- debug layout issues  
- work with complex templates  

---

### Important notes

- This is a visual aid only  
- It does not affect the generated document

---

## When to use Editor Tools

Use Editor Tools when you need to:

- format text and layout  
- adjust visual structure  
- refine the appearance of your document
