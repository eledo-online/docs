# Lists

Lists allow you to organize content into structured sequences.

---

## Where to find Lists

List options are available in the editor toolbar.

<ImageWithCaption
  src="/img/product/template-editor/editor-tools/lists-toolbar.png"
  alt="List toolbar icons"
  caption="Numbered and bullet list options in the toolbar."
/>

---

## Available types

### Bullet list

Creates an unordered list using bullet points.

---

### Numbered list

Creates an ordered list with numbered items.

---

## How it works

- Place the cursor where you want to start the list  
- Click the desired list type  
- Start typing list items  
- Press **Enter** to create a new item  

---

## When to use Lists

Use Lists when you need to:

- organize content clearly  
- present steps or sequences  
- improve readability  

---

## Important notes

- Lists follow the same cursor-based behavior as text  
- You can combine lists with indentation to create nested structures
