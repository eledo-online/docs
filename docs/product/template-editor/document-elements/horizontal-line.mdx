# Horizontal Line

A Horizontal Line inserts a visual separator into your document.

---

## Add a Horizontal Line

To insert a horizontal line into your document:

1. Place the cursor where you want the line to appear  
2. Click the **Horizontal Line** icon in the toolbar  

The line will be inserted at the cursor position.

<ImageWithCaption
  src="/img/product/template-editor/document-elements/horizontal-line-toolbar.png"
  alt="Horizontal Line icon in toolbar"
  caption="Insert a horizontal line from the toolbar."
/>

---

## How it works

- The line spans across the document width  
- It acts as a visual divider between sections  
- No additional configuration is required  

---

## When to use Horizontal Line

Use a Horizontal Line when you need to:

- separate sections visually  
- improve readability  
- structure content without creating a new page
