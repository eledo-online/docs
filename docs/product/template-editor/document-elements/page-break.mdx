# Page Break

A Page Break forces the content that follows to start on a new page.

---

## Add a Page Break

To insert a page break into your document:

1. Place the cursor where you want the new page to begin  
2. Click the **Page Break** icon in the toolbar  

The page break will be inserted at the cursor position.

<ImageWithCaption
  src="/img/product/template-editor/document-elements/page-break-toolbar.png"
  alt="Page Break icon in toolbar"
  caption="Insert a page break from the toolbar."
/>

---

## How it works

- Content before the break stays on the current page  
- Content after the break starts on a new page  
- No additional configuration is required  

---

## When to use Page Break

Use a Page Break when you need to:

- separate sections of a document  
- control pagination  
- ensure specific content starts on a new page
