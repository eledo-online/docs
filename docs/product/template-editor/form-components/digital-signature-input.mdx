# Digital Signature Input

A Digital Signature Input allows you to add a signature field into your PDF document.

The user can draw or provide their signature directly in the document.

![Digital Signature Input Configuration](/img/product/template-editor/form-components/digital-signature-input-configuration.png)

---

## Add a Digital Signature Input

To insert a Digital Signature Input into your document:

1. Place the cursor where you want the signature field to appear  
2. Click the **Form** dropdown in the toolbar  
3. Select **Digital signature input**  
4. Configure the field and confirm  

The signature field will be inserted at the cursor position.

![Digital Signature Input in Form dropdown](/img/product/template-editor/form-components/digital-signature-input-toolbar.png)
<figcaption>*Select **Digital signature input** from the Form dropdown.*</figcaption>

---

## Configuration

### Name (expression)

Defines the identifier of the signature field.

- Can be a static name or an expression  
- Evaluated during document generation  
- Useful when generating multiple signature fields dynamically  

---

### Width

Defines the width of the signature field.

- Accepts values with units (e.g. `100px`)  
- Controls how much space is available for the signature  

---

### Height

Defines the height of the signature field.

- Accepts values with units (e.g. `40px`)  
- Controls the vertical space for the signature  

---

## When to use Digital Signature Input

Use a Digital Signature Input when you need:

- user-provided signatures  
- document approval or confirmation  
- legally or visually signed documents
