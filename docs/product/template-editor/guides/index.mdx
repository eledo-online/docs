# Introduction

This section helps you learn how to use the **Template Editor**.

Guides are designed to walk you through common tasks and patterns — from simple text rendering to more advanced layouts.

---

## How to use these guides

If you are new to the Template Editor, follow this order:

1. **Printing Text** — understand how content is rendered  
2. **Expressions** — learn how to work with dynamic values  
3. **Formatting Numbers / Dates** — control how data is displayed  
4. **Components** — explore available building blocks  

---

## What you will learn

These guides cover:

- inserting and formatting content  
- working with dynamic data  
- building layouts  
- combining components to solve real use cases  

---

## Guides vs Components

- **Guides** show *how to use features together*  
- **Components** describe *how individual features work*  

If you need detailed configuration, refer to the Components section.
