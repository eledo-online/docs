# Filling in PDF Forms

Eledo supports working with existing PDF forms directly inside [monday.com](https://monday.com/) workflows.

You can either:

- populate fields that already exist in a PDF form
- create entirely new interactive form fields inside your template

This section explains both approaches and how to use them in document automation workflows.

PDF forms are especially useful for:

- contracts
- government documents
- onboarding forms
- standardized templates
- administrative workflows
