# Create Image Field

To create an image field in a PDF form, follow these steps:

- In the Eledo editor, select the **Image Field** tool  
- Drag to draw the field in your document  
- Set a **Field ID** (letters, numbers, and underscores only; no spaces)  
- In **Data (expression)**, select an image source (for example: `item.files.public_url`)  
- Confirm and apply the configuration  

![image](/img/integrations/monday/MondayPDFFormDynamicImageField.gif)

---

## Notes

The image source must be a publicly accessible URL.
