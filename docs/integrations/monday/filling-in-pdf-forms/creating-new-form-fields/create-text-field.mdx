# Create Text Field

To create a text form field in a PDF form, follow these steps:

- In the Eledo editor, select the [**Text Field**](/docs/product/template-editor/form-components/text-field.mdx) tool  
- Drag to draw the field in your document  
- Set a **Field ID** (letters, numbers, and underscores only; no spaces)  
- In **Data (expression)**, select a value (for example: `item.name`) or use an expression  
- Confirm and apply the configuration
