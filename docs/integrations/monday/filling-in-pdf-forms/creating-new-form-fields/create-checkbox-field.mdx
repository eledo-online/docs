# Create Checkbox Field

To create a checkbox field in a PDF form, follow these steps:

- In the Eledo editor, select the [**Checkbox Field**](/docs/product/template-editor/form-components/checkbox.mdx) tool  
- Place the field into your document  
- Set a **Field ID** (letters, numbers, and underscores only; no spaces)  
- In **Data (expression)**, select a value (for example: `item.checkbox`) or use an expression  
- Confirm and apply the configuration  

![image](/img/integrations/monday/MondayPDFFormCheckboxFormField.gif)

---

## Notes

Checkbox fields require a boolean value (`true` or `false`).

You can use Eledo functions or conditions to generate the value.
