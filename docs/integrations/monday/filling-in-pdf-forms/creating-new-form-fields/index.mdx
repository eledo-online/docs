# Creating New Form Fields

This section explains how to create new interactive PDF form fields directly inside the Eledo editor.

Unlike populating existing PDF forms, these guides focus on building form fields from scratch within your template.

You will learn how to create:

- text fields
- image fields
- checkbox fields
- QR code fields

These fields can then be populated dynamically using monday.com data.

Typical use cases include:

- interactive contracts
- editable forms
- onboarding documents
- reusable PDF templates
- machine-readable workflows
