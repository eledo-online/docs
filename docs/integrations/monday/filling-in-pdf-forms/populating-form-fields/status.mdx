# Populating Text Field with a Status

To print a value ​​in PDF Form from a Status Column from your monday.com, follow these steps:  
- In Eledo editor click on a custom form field and configuration window will appear.
- Click into **Data (expression)** field to open Data Expression builder.
- Find and select a Status Column using the Fields navigation menu. Start with the **item** and continue with **status**, and **label** for example.
- Confirm twice and you're done!
