# Populating Status

To populate a PDF form field using a value from a Status column in your monday.com board, follow these steps:

- In the Eledo editor, select the form field  
- Click into **Data (expression)** to open the expression builder  
- Select a status field (for example: `item.status.label`)  
- Confirm and apply the value
