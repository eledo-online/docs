# Populating Form Fields

This section explains how to populate existing PDF form fields using data from monday.com boards.

The guides demonstrate how to map monday.com column values into interactive PDF form fields such as:

- text fields
- number fields
- status values
- dates
- checkboxes
- dropdowns

This approach is useful when working with pre-existing PDF forms that already contain editable fields.

Typical use cases include:

- contracts
- onboarding forms
- administrative documents
- government forms
- reusable business templates
