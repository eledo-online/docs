# Create Image Field

To insert a Dynamic Image ​​into your PDF Form, follow these steps:  
- In Eledo editor click on a Dynamic Image tool, then drag to draw a rectangle and create a new form field. Confirm the shape and configuration window will appear.
- Field ID is a unique identifier of your field. It may contain only letters, numbers, underscore character. No space is allowed.
- For Data (expression) field find and select a File Column using the Fields navigation menu, for example **item**, **files** and **public url**.
- Confirm twice and you're done!
