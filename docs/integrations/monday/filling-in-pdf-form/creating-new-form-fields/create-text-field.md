# Create Text field

To create a text form field ​​in PDF Form, follow these steps:  
- In Eledo editor click on a Text Box tool, then drag to draw a rectangle and create a new form field. Confirm the shape and configuration window will appear.
- Field ID is a unique identifier of your field. It may contain only letters, numbers, underscore character. No space is allowed.
- For Data (expression) field find and select a Text Column using the Fields navigation menu. You can also use some of [Eledo functions](/docs/product/data-expressions/general-functions.md), e.g. `TODAY()`. 
- Confirm twice and you're done!
