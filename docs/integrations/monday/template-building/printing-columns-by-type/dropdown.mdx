# Printing Dropdown Column

To print a value ​​from a Dropdown Column from your monday.com board, follow these steps:  
- In Eledo editor click on a **Text box** component from top toolbar. Configuration window will appear.
- Click into **Data (expression)** field to open Data Expression builder.
- Find and select a dropdown column using the Fields navigation menu. Start with the **item** and continue with **dropdown**.
- If you need to format this value, try one of our [array functions ](/docs/product/data-expressions/array-functions.mdx)
- Confirm twice and you're done! Dynamic text component will be placed at cursor position.
