# Printing Long Text Column

To print a value from a Long Text column in your monday.com board, follow these steps:

- In the Eledo editor, add a **Text Box** component  
- Click into **Data (expression)** to open the expression builder  
- Select a long text field (for example: `item.long_text`)  
- Confirm and place the value into the document
