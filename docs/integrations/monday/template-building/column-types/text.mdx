# Printing Text Column

To print a value from a Text column in your monday.com board, follow these steps:

- In the Eledo editor, add a [**Text Box**](/docs/product/template-editor/components/text-box.mdx) component  
- Click into **Data (expression)** to open the expression builder  
- Select a text field (for example: `item.name`)  
- Optionally format the value using [text functions](/docs/product/template-editor/data-expressions/text-functions.mdx)  
- Confirm and place the value into the document
