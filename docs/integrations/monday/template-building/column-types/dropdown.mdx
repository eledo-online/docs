# Printing Dropdown Column

To print a value from a Dropdown column in your monday.com board, follow these steps:

- In the Eledo editor, add a **Text Box** component  
- Click into **Data (expression)** to open the expression builder  
- Select a dropdown field (for example: `item.dropdown`)  
- Optionally format the value using [array functions](/docs/product/template-editor/data-expressions/array-functions.mdx)  
- Confirm and place the value into the document  

![image](/img/integrations/monday/MondayPrintDropdownColumn.gif)

---

## Notes

Dropdown values may contain multiple selections.

You can use array functions to control how they are displayed.
