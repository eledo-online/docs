# Column Types

This section explains how to print values from supported monday.com column types using Eledo templates.

Each guide demonstrates how to:

- access column values
- configure data expressions
- format output correctly
- work with monday.com-specific structures

These column guides form the foundation of monday.com document generation workflows.

---
