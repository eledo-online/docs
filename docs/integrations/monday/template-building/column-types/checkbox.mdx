# Printing Checkbox Column

To print a value from a Checkbox column in your monday.com board, follow these steps:

- In the Eledo editor, add a [**Dynamic Checkbox**](/docs/product/template-editor/components/dynamic-checkbox.mdx) component  
- Click into **Data (expression)** to open the expression builder  
- Select a checkbox field (for example: `item.checkbox`)  
- Confirm and place the checkbox into the document
