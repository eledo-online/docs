# Template Building

This section explains how to build Eledo templates using data from [monday.com](https://monday.com/) boards.

The guides focus on practical document generation patterns commonly used with monday.com integrations.

You will learn how to:

- print monday.com column values
- work with subitems
- generate QR codes
- print images from Files columns
- use connected board items
- filter and organize repeated content

These patterns can be combined to create invoices, reports, labels, contracts, catalogs, and other automated PDF documents.
