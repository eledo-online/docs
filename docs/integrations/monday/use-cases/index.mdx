# Use Cases

This section contains practical monday.com workflow examples and template patterns built with Eledo.

The guides demonstrate how to solve common real-world document generation scenarios using monday.com boards, items, subitems, and connected data.

You will learn how to:

- generate multiple PDFs automatically
- work with subitems
- build grouped document layouts
- generate QR codes linked to monday.com items
- create image-based document layouts

---

# Available Use Cases

## Creating PDFs for Each Subitem Separately

Generate one PDF document for each subitem on a monday.com board.

Useful for:

- work orders
- labels
- packing slips
- task-specific documents

→ [Creating PDFs for Each Subitem Separately](./creating-pdfs-for-each-subitem-separately.mdx)
