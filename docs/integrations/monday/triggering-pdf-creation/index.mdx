# Triggering PDF Creation

This section explains how to trigger PDF generation workflows from monday.com.

Eledo integrates with monday.com automation recipes and webhooks to generate documents automatically based on board activity.

Typical triggers include:

- item creation
- status changes
- button clicks
- form submissions
- custom webhook events

These guides help you connect monday.com events with Eledo document generation workflows.
