# Guides

This section contains practical guides for integrating Eledo with automation platforms.

The guides are organized into two main categories:

- [**Template Building**](./template-building/index.mdx) — how to prepare Eledo templates for automation workflows
- [**Scenarios**](./scenarios/index.mdx) — complete automation examples for generating and processing PDF documents

These guides are designed to help you move from template creation to fully automated document generation.

---

# Available Categories

## Template Building

Learn how to prepare templates for automated PDF generation.

Topics include:

- [printing text](./template-building/printing-text.mdx), [numbers](./template-building/printing-number.mdx), and [dates](./template-building/printing-date.mdx)
- [images](./template-building/printing-image.mdx) and [QR codes](./template-building/printing-qr-code.mdx)
- [dynamic checkboxes](./template-building/printing-checkbox.mdx)
- [repeating rows and structured data](./template-building/repeating-table-row.mdx)

→ [Template Building](./template-building/index.mdx)

Explore complete automation scenarios using Eledo.
