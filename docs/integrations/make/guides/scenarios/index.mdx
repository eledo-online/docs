# Scenarios

This section contains practical [Make](https://www.make.com/) automation scenarios built with Eledo.

The guides demonstrate how to connect Eledo with external services to automate PDF generation and document processing.

You will learn how to:

- generate PDFs automatically
- populate existing PDF forms
- connect Eledo with cloud services and databases
- build end-to-end automation scenarios
