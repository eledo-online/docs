# Repeating Table Row

This guide shows how to print a list of items into a table using [Make](https://www.make.com/).

---

## Step 1 — Prepare your template

In the Eledo editor:

1. Create a [table](/docs/product/template-editor/document-elements/table.mdx) with:
   - one row for headers  
   - one row for data  

2. Place the cursor inside the second row  

3. Click [**Block Repetition**](/docs/product/template-editor/components/block-repetition.mdx)

4. Click **Select Parent Block** to select the entire table row  

5. In **Data (expression)**, enter the list name (for example `Products`)  

This will repeat the table row for each item in the list.

---

### Add dynamic values

Inside the repeated row:

1. Add [**Text Box (TXT)**](/docs/product/template-editor/components/text-box.mdx) components  
2. Use field names for item properties (for example `Name`, `Quantity`, `DeliveryDate`)  

![Repeating table row configuration](/img/integrations/make/RepeatingTableRowMake1.gif)

:::tip

Learn more about Block Repetition:  
→ [/docs/product/template-editor/components/block-repetition.mdx](/docs/product/template-editor/components/block-repetition.mdx)

:::

---

## Step 2 — Use the template in Make

In your Make scenario:

1. Add the **Eledo PDF** module  
2. Select your template  
3. Click **Refresh Custom Fields**

---

## Step 3 — Map your data

1. Enable **Map** on the list field (for example `Products`)  
2. Select **Array**  
3. Open the mapping configuration  
4. Map fields (for example `Name`, `Quantity`, `DeliveryDate`)  

![Mapping table rows](/img/integrations/make/RepeatingTableRowMake2.gif)

---

## Result

When the scenario runs:

- each item in the list is processed  
- the table row is repeated  
- values are inserted into each column
