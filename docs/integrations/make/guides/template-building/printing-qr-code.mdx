# Printing QR Code

This guide shows how to print QR codes into your PDF using Make.

---

## Step 1 — Prepare your template

In the Eledo editor:

1. Click **Barcodes 2D** → select **QR Code**  
2. In **Data (expression)**, enter a field name (for example `ProductQR`)  
3. Confirm and save the template  

![QR code configuration](/img/integrations/make/PrintingQRCodeMake1.gif)

:::tip

Learn more about QR codes:  
→ [/docs/product/template-editor/guides/generating-qr-codes.mdx](/docs/product/template-editor/guides/generating-qr-codes.mdx)

:::

---

## Step 2 — Use the template in Make

In your Make scenario:

1. Add the **Eledo PDF** module  
2. Select your template  
3. Click **Refresh Custom Fields**

Your field (for example `ProductQR`) will appear.

---

## Step 3 — Map your data

1. Click into the field (for example `ProductQR`)  
2. Provide the value to encode (for example a URL or text)  
3. Confirm the mapping  

![Mapping QR code value](/img/integrations/make/PrintingQRCodeMake2.gif)

---

## Important notes

- QR codes encode text — the format of the value determines behavior  
- Common use case: encode a URL  

Example:

```
https://example.com/product/123
```

---

## Result

When the scenario runs:

- the value is passed from your source  
- Eledo encodes it into a QR code  
- the QR code is rendered in the document
