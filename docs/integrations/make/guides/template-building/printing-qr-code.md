# Printing QR Code

To print QR code ​​from make.com, follow these steps:  
- In Eledo editor click on **Barcodes 2D** and select **QR Code ** component from top toolbar. Configuration window will appear. 
- Click into **Data (expression)** field to open Data Expression builder.
- Type the word you want to represent the QR code value. It will be used as input field name. This word cannot contain a space nor special characters, just letters, numbers and underscore. For example **ProductQR**.
- Confirm twice and you're done! Dynamic QR Code will be placed at cursor position.
- Save the template and input field will be automatically created from the data expression.

![image](/img/integrations/make/PrintingQRCodeMake1.gif)

- In your make.com scenario click on Eledo modul and refresh data from the template.
- **ProductQR** input field should be now available to be mapped with a date value.
- You can compose your product URL and use it as a QR code content for example.
- Confirm and you're done!

![image](/img/integrations/make/PrintingQRCodeMake2.gif)
