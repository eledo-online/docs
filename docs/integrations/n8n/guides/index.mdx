# Guides

This section contains practical guides for integrating Eledo with automation platforms.

The guides are organized into two main categories:

- [**Template Building**](./template-building/index.mdx) — how to prepare Eledo templates for automation workflows
- [**Workflows**](./workflows/index.mdx) — complete automation examples for generating and processing PDF documents

These guides are designed to help you move from template creation to fully automated document generation.

---

# Available Categories

## Template Building

Learn how to prepare templates for automated PDF generation.

Topics include:

- printing text, numbers, and dates
- images and QR codes
- dynamic checkboxes
- repeating rows and structured data

→ [Template Building](./template-building/index.mdx)

## Workflows

Explore complete automation workflows using Eledo.

Typical examples include:

- generating PDFs from forms or spreadsheets
- sending PDFs by email
- uploading documents to cloud storage
- processing multiple records automatically

→ [Workflows](./workflows/index.mdx)
