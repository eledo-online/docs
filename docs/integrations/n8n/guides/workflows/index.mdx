# Workflows

This section contains practical n8n workflow examples built with Eledo.

The guides demonstrate how to automate PDF generation and document processing using n8n workflows and Eledo templates.

You will learn how to:

- generate PDFs automatically
- populate existing PDF forms
- work with structured JSON payloads
- build advanced automation workflows
