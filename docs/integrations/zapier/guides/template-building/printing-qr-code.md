# Printing QR Code

To print QR code ​​from make.com, follow these steps:  
- In Eledo editor click on **Barcodes 2D** and select **QR Code ** component from top toolbar. Configuration window will appear. 
- Click into **Data (expression)** field to open Data Expression builder.
- Type the word you want to represent the value. This word cannot contain a space. For example **ProductQR**.
- Confirm twice and you're done! Dynamic QR Code will be placed at cursor position.
- Don´t forget to Save the template.
