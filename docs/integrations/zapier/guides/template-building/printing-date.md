# Printing Date

To print a date value ​​from make.com, follow these steps:
- In Eledo editor click on a **Text box** component from top toolbar. Configuration window will appear. 
- Click into **Data (expression)** field to open Data Expression builder.
- Type the word you want to represent the value. This word cannot contain a space. For example **DeliveryDate**.
- If you need to format this value, try one of our [DATE functions](/docs/product/template-building-guides/formatting-dates.md)
- Confirm twice and dynamic text component will be placed at cursor position.
- Don´t forget to Save the template.
