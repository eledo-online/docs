# Printing Text

To print a text value ​​from make.com, follow these steps:
- In Eledo editor click on a **Text box** component from top toolbar. Configuration window will appear. 
- Click into **Data (expression)** field to open Data Expression builder.
- Type the word you want to represent the value. This word cannot contain a space. In our case **ProductName**.
- If you need to format this value, try one of our [text formatting functions](/docs/product/template-editor/data-expressions/text-functions.mdx)
- Confirm twice and dynamic text component will be placed at cursor position.
- Don´t forget to Save the template.
