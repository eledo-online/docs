# User Profile

Returns information about the [authenticated](./authentication.mdx) user.

- **HTTP Method:** `GET`  
- **Endpoint:** `https://eledo.online/api/RESTv1/Profile`

---

## Response

### Body

| Name      | Type   | Description                          |
|-----------|--------|--------------------------------------|
| `account` | String | Account identifier (email)           |

---

## Example

```
{
    "account": "user@example.com"
}
```

---

## When to use User Profile

Use this endpoint to:

- verify that authentication is working  
- retrieve the current account identifier
