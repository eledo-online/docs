# User Profile

Returns the user's profile.

- **HTTP Method:** `GET`  
- **Endpoint:** `https://eledo.online/api/RESTv1/Profile`

---

### Response Body Parameters

| Name      | Type   | Description                          |
|-----------|--------|--------------------------------------|
| `account` | String | Account's login identifier (email)   |
