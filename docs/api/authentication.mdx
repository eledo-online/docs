# Authentication

Currently, we are using **HTTP Header Authentication** with an **API Key**.

### Authentication Headers

| Header   | Type   | Required | Description        |
|----------|--------|----------|--------------------|
| Api-Key  | String | Yes      | Use your API Key   |
