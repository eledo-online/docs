# Printing Column from Connected Item

To print a value ‚Äã‚Äãfrom a Connected Item from your monday.com board, follow these steps:
- In Eledo editor click on a **Text box** component from top toolbar. Configuration window will appear. 
- Click into **Data (expression)** field to open Data Expression builder.
- Find and select a text column from connected board using the Fields navigation menu. Start with the **item**, continue with **connect_boards** and then select **text** for example.
- If you need to format this value, try one of our [text formatting functions](/_inbox/40-data-expressions/30-text-functions.md)
- Confirm twice and you're done! Dynamic text component will be placed at cursor position.
