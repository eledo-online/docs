# Populating Text

To populate a Text field ‚Äč‚Äčin PDF Form with a value from a Text Column from your monday.com board, follow these steps:  
- In Eledo editor click on the Text field and configuration window will appear.
- Click into **Data (expression)** field to open Data Expression builder.
- Find and select a Text Column using the Fields navigation menu. Start with the **item** and continue with **name** for example.
- If you need to format this value, try one of our [text formatting functions](https://eledo.online/documentation/data_expressions/text_functions)
- Confirm twice and you're done! Dynamic text component will be placed at cursor position.
