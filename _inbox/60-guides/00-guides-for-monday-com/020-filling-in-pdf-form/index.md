# Filling in PDF Form
Quick guides to help you with PDF Form filling using Eledo in monday.com

## Populating Form fields
Find out how you can populate existing Form Fields in your PDF Form. Numbers and dates formatting, checkboxes, dropdowns and even images.

- **[Text](/_inbox/60-guides/00-guides-for-monday-com/020-filling-in-pdf-form/00-populating-form-fields/010-text.md)**  
Populate Text field with a plain text

- **[Number](/_inbox/60-guides/00-guides-for-monday-com/020-filling-in-pdf-form/00-populating-form-fields/020-number.md)**  
Populate Text field with a number and optionaly format it as a currency or a number with specific number of decimal places

- **[Date](/_inbox/60-guides/00-guides-for-monday-com/020-filling-in-pdf-form/00-populating-form-fields/40-date.md)**  
Populate Text field with a date and format it into custom date format

- **[Checkbox](/_inbox/60-guides/00-guides-for-monday-com/020-filling-in-pdf-form/00-populating-form-fields/50-checkbox.md)**  
Populate Checkbox field from a Checkbox Column or others like Status, Number and more.

- **[Dropdown](/_inbox/60-guides/00-guides-for-monday-com/020-filling-in-pdf-form/00-populating-form-fields/60-dropdown.md)**  
Populate Dropdown field from a Status column and map status labels to dropdown options if they are different

- **[Create Checkbox field](/_inbox/60-guides/00-guides-for-monday-com/020-filling-in-pdf-form/20-creating-new-form-fields/30-create-checkbox-field.md)**  
Add new checkbox field and populate it with value from a Checkbox column
