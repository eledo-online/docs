# Getting Started with Eledo

With the Eledo web service, you can **create PDF** documents from a predefined **template**. We provide both an Automatic Web Form and a REST API, so you can focus solely on the template.

Let's start with a **generated PDF** to demonstrate basic features:

[Open Example PDF](https://eledo.online/documents/index:downloadpreview/example_introduction?t:cp=create)

Once you’ve read the document, you might be curious how the template for it looks. Check it out in our **online template editor**:

[Open Template Editor](https://eledo.online/deditor/create/example_introduction)

Yes, all you need is a template. Eledo automatically prepares a **Web Form** and a REST API for you:

[Open Web Form](https://eledo.online/create/example_introduction)

Sometimes it’s time-consuming to create a document template from scratch. That’s why we’re building a **public document library** to help you get started. You can copy a public document template to personalize it, or use it directly:

[Open Public Library](https://eledo.online/documents)
