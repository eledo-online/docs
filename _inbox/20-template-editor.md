# Eledo Template Editor

The **Eledo Template Editor** supports the creation and editing of Eledo templates, which are used during integration for Eledo PDF generation.

You can launch the Template Editor directly from the **Templates Library** by creating a new template or editing an existing one.

Learn more about the editor’s features below.

---

## Menu

The **Menu** contains functionality familiar from standard text editors, along with Eledo Dynamic Components used to create and edit dynamic parts of your documents.

Dynamic components can be fully customized using **Expressions**.

---

## Main Document

Add your core text content and dynamic components here.

---

## Input Fields

Import data structures or retrieve them directly from your data source to make template building easier and more efficient.

---

## Background Document

This option allows you to edit headers and footers of your document templates, including elements such as page numbers and labels.

---

## Settings

Here you can configure:

- File name  
- Format  
- Localization  
- Template description  

---

## Document Security

You can define document and owner passwords here.

Additional options include:

- Printing permissions  
- Modification permissions  
- Extra content settings
