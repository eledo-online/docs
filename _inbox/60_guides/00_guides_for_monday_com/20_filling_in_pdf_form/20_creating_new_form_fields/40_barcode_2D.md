# Create QR code field

To insert a QR code ‚Äã‚Äãinto your PDF Form, follow these steps:  
- In Eledo editor click on a Barcode 2D tool, and configuration window will appear.
- Field ID is a unique identifier of your field. It may contain only letters, numbers, underscore character. No space is allowed.
- For Data (expression) field find and select any column you want to refer to using the Fields navigation menu. See a quick tutorial on how to create a [QR code with a link to an item](https://eledo.online/documentation/guides/monday_com/use_cases/qr_code_with_link_to_monday_com_item).
- Click on Place button and place the QR code field into your document. 
- Confirm twice and you're done!
