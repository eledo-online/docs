# Filling in PDF Form
Quick guides to help you with PDF Form filling using Eledo in monday.com

## Populating Form fields
Find out how you can populate existing Form Fields in your PDF Form. Numbers and dates formatting, checkboxes, dropdowns and even images.

- **[Text](/_inbox/60_guides/00_guides_for_monday_com/020_filling_in_pdf_form/00_populating_form_fields/010_text.md)**  
Populate Text field with a plain text

- **[Number](/_inbox/60_guides/00_guides_for_monday_com/020_filling_in_pdf_form/00_populating_form_fields/020_number.md)**  
Populate Text field with a number and optionaly format it as a currency or a number with specific number of decimal places

- **[Date](/_inbox/60_guides/00_guides_for_monday_com/020_filling_in_pdf_form/00_populating_form_fields/40_date.md)**  
Populate Text field with a date and format it into custom date format

- **[Checkbox](/_inbox/60_guides/00_guides_for_monday_com/020_filling_in_pdf_form/00_populating_form_fields/50_checkbox.md)**  
Populate Checkbox field from a Checkbox Column or others like Status, Number and more.

- **[Dropdown](/_inbox/60_guides/00_guides_for_monday_com/020_filling_in_pdf_form/00_populating_form_fields/60_dropdown.md)**  
Populate Dropdown field from a Status column and map status labels to dropdown options if they are different

Add new form fields to your PDF and populate them with data from your monday.com board.

Add new Text field and populate it with a plain text

- **[Create Image field](/_inbox/60_guides/00_guides_for_monday_com/020_filling_in_pdf_form/20_creating_new_form_fields/20_create_image_field.md)**  
Add new Image field and populate it with image from a Files column

- **[Create Checkbox field](/_inbox/60_guides/00_guides_for_monday_com/020_filling_in_pdf_form/20_creating_new_form_fields/30_create_checkbox_field.md)**  
Add new checkbox field and populate it with value from a Checkbox column
